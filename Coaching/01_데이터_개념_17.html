Chapter 3. 고용노동부의 데이터

일하는 방식의 문제: 인식

현장 중심의 조직문화

오랜 기간 현장 해결 중심으로 업무가 정착되었습니다.
당장의 문제 해결이 급하다 보니 데이터 기록과 관리의 중요성을 상대적으로 낮게 인식합니다.

"문제 없으면 그만" 관행

현장에서 당장 민원이 없거나 사고가 터지지 않으면,
데이터 관리가 다소 부실하더라도 문제 삼지 않는 분위기가 존재합니다.

시스템 대신 수기 취합

국회나 언론 대응 시, 내부 통계 시스템을 신뢰하거나 활용하기보다
그때그때 공문이나 메일로 엑셀 파일을 취합하는 방식을 당연하게 여깁니다.

데이터 기반 행정의 걸림돌

발표자 노트

  • 고용노동부의 데이터 문제는 단순히 시스템 문제가 아닙니다. 일하는 방식, 더 깊게는 '인식'의 문제입니다.
  • 첫째, 현장 중심의 조직문화가 강합니다. 당장의 민원 해결과 현장 단속이 최우선이다 보니, 그 과정을 꼼꼼하게 데이터로 남기는 일은 '부수적인 행정 업무'로 치부되곤 합니다.
  • 둘째, 당장 사고가 나지 않으면 데이터 부실을 문제 삼지 않습니다. 데이터 품질이 나빠도 당장 업무가 돌아가면 넘어가는 것이죠.
  • 셋째, 가장 큰 문제는 '수기 취합' 관행입니다. 급한 국회 요구자료가 오면 시스템을 열어보는 게 아니라, 전국의 지청에 "엑셀로 언제까지 보내라"고 공문을 보냅니다.
  • 이러한 방식은 데이터가 시스템에 축적되는 것을 막고, 담당자의 피로도만 높이는 악순환을 만듭니다.
