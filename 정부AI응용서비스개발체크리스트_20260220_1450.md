## 4단계: As-Is / To-Be 정의

### 4-1. As-Is (현재 상태)

**현재 업무 흐름 설명**

```
1) 실업급여 신청 -> 2) 구직활동 -> 3) 구직활동 인정 -> 4) 급여지급 결정 -> 5) 급여지급
```

**소요 시간**

```
평균 1~2일
```

**병목 지점**

```
서류검토 단계에서 1일, 결재단계에서 1일 소요되며 실업급여 신청이 집중되는 월말, 연말에는 병목현상 발생
```

**사람 의존 구간**

```
구직활동의 적정성 판단은 지방고용노동관서 실업급여 담당자가 전결로 결정
```

### 4-2. To-Be (개선 후)

**AI 도입 후 업무 흐름**

```
1) 민원 접수 2) 구직활동 3) AI가 구직활동의 적정성 판단 4) 적정한 경우 - 자동지급 / 부적정한 경우 - 담당자 재검토 5) 재검토 결과에 따라 구직급여 지급여부 결정 6) 급여 지급
```

**자동화/보조/판단 분리**

```
보조(AI 구직활동의 적정성 여부를 판단)
```

**사람의 역할 재정의**

```
담당자는 AI가 판단한 부지급 건에 대한 2차 검토 실시, 아울러 지급 결정된 건에 대해 일정 확률로 2차 검토 실시
```
